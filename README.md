# Модная разработка Яцышина и Чепусова




### Установка библиотек

```pip install -r requirements.txt```


###  Создание экзешника 
 ```pyinstaller --onefile --noconsole main.py```

### Импорт либ в файл

```pip freeze > requirements.txt```

### Обфускация

```pyarmor gen name.py```
